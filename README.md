# Nearnest App

Nearnest is a minimal, modern Flutter app for iOS and Android that helps users discover and contact trusted local vendors for urgent services like photography, home decoration, catering, makeup, and event planning.

## Highlights

- Cross-platform Flutter UI using Material 3.
- Search, category chips, budget filtering, verified vendor cards, and vendor details.
- Secure-by-default contact flow using validated `tel:`/`mailto:` links with no hard-coded secrets or sensitive local storage.
- Booking enquiry form with client-side validation and privacy-first copy.

## Getting started

```bash
flutter pub get
flutter run
```

## Tests

```bash
flutter test
```
